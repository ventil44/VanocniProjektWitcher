# The Witcher IV *leaked*

This is The Witcher IV in its earliest stage, so its just a text game. Coded in java, people often say that they are a changed person after completing this game.

## Functions

- Moving from one location to another
- Turned based combat system
- Talking with NPCs
- Quests
- Taking items into inventory
- Healing with potions
- Equipping weapons

## Commands

- **attack**: Attack enemies in the current location, if there are enemies
- **exit**: Exits the game
- **go**: Go to one of the connected locations from your current one
- **heal**: Heal yourself with a healing potion, if u have one
- **help**: Displays all available commands
- **inventory**: Displays items in your inventory
- **quest**: Starts a quest in the current location, if theres one
- **status**: Displays your current HP
- **take**: Take an item from the current location, if theres any to take
- **talk**: Talk to an NPC in the current location, if there is someone to talk to

## How to run

1) Extract the .zip
2) Make sure .jar and .bat are in the same folder
3) Double click run.bat

--- IF URE CLONING MY REPOSITORY ---
1) Clone the repository
2) Open the project in IntelliJ IDEA
3) Build an artifact to make it a .jar file (use youtube for that)
4) Create a .txt file and type "java -jar .\VanocniProjektWitcher.jar"
5) Make the .txt file a .bat file
6) Make sure .jar and .bat are in the same folder
7) Double click run.bat

## Requirements

Java Runtime Environment (JRE) 17 or newer.
Windows OS to run .bat
